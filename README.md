# NextJS 14 Template

**Features**

1. Tailwind Included.
2. Standard commit messages using CommitLint
3. Format and Lint checks before commit
4. Build check before push
5. Customized eslint configuration

**Includes**

1. NextJS 14
2. Tailwind 3.3
3. Typescript 5
4. Eslint 8
5. Prettier 3.3
6. Husky 8
7. CommitLint 18

**Requires**
Node 20 and above is recommended.

**Notes**
Uses src and app directory.
